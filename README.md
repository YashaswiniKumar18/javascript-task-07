# JavaScript Date, Async, Promise & Fetch Practice

This project contains JavaScript practice tasks covering:

- Date Object
- setTimeout & setInterval
- Promises
- Fetch API
- JavaScript Execution Order

---

## Tasks Covered

### Section 1 – Date Object
- Custom Digital Clock
- Find Current Day
- Age Calculator
- Create Specific Date

### Section 2 – Async Timing
- Delayed Message using setTimeout
- Stop Interval using clearInterval

### Section 3 – Promises
- Resolve & Reject based on condition
- Using .then(), .catch(), .finally()

### Section 4 – Fetch API
- Fetch product data
- Display title and price

### Section 5 – Execution Order
Understanding JavaScript event loop and asynchronous behavior.
